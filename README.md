## Command

```
bash -c "$(curl -fsSL raw.github.com/kurarrr/dotfiles/master/setup.sh)"
# Download dotfiles
./setup.sh initialize
# initialize mac configurations and brew (and brew-cask) apps
./setup.sh deploy
# add symbolic link
```
